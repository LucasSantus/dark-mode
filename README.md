<h3 id="sobre">:information_source: Sobre</h3>

> Este projeto foi desenvolvido com intuito de adicionar opções de temas dark & light viabilizando a satisfação do usuário, foram utilizadas as tecnologias HTML5, CSS3 e JavaScript.

A ideia é:

_"Criar uma funcionalidade de dark mode viabilizando a troca de temas entre light & dark"_
